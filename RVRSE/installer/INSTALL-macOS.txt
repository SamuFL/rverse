RVRSE — Installation Guide (macOS)
-----------------------------------

Thank you for downloading RVRSE!

1. Open the RVRSE `.pkg` installer.

2. Follow the installer prompts. By default, RVRSE installs to:

  AU:    /Library/Audio/Plug-Ins/Components/RVRSE.component
  VST3:  /Library/Audio/Plug-Ins/VST3/RVRSE.vst3
  CLAP:  /Library/Audio/Plug-Ins/CLAP/RVRSE.clap
  App:   /Applications/RVRSE.app
  Demo samples: /Library/Application Support/RVRSE/Examples/

3. Relaunch your DAW and rescan plugins if needed.


macOS Security (Gatekeeper)
---------------------------
Official RVRSE release installers are signed with Apple Developer ID,
notarized by Apple, and stapled before distribution.

No `xattr` workaround should be required for the official installer.

If Gatekeeper still warns after a fresh release download, please report
it at:

  https://github.com/SamuFL/rverse/issues
