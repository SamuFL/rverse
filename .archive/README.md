# Archive

This directory holds **historical project artifacts that are no longer part of the active workflow**.

## Beads historical snapshot

The `beads-historical/` directory is the archived Beads tracker database that was used during
RVRSE's earlier development. It is kept **read-only for historical reference** so future
contributors can inspect past planning and decision context without treating it as the live source
of truth.

Active project management now happens in:

- [GitHub Issues](https://github.com/SamuFL/rverse/issues)
- [v1.1.0 milestone](https://github.com/SamuFL/rverse/milestone/1)
- [RVRSE Development project board](https://github.com/users/SamuFL/projects/1)
